# DataCleaning
Explore common data quality issues such as missing values, duplicates, and inconsistencies, and use SQL queries to clean and transform data into usable formats. 

Functions and Queries Used:

1) ALTER                  
2) UPDATE
3) Typecasting            
4) NULL/ISNULL
5) SUBSTRING              
6) CHARINDEX
7) PARSENAME      
8) REPLACE
9) CASE WHEN           
10) Subqueries
11) ROW_NUMBER         
12) RANK
